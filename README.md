# 🍔 Meu Pedido Web

Uma Landing Page moderna para estabelecimentos de alimentação com cardápio online, carrinho de compras e envio automático do pedido pelo WhatsApp.

O projeto foi desenvolvido pensando em pequenos negócios que desejam oferecer uma experiência simples e profissional aos seus clientes, sem a necessidade de um sistema complexo.

# 🎯 Objetivo do projeto

Este projeto nasceu com o objetivo de criar uma solução reutilizável para pequenos estabelecimentos que desejam vender pela internet utilizando apenas o WhatsApp.

A ideia é permitir que diferentes negócios utilizem a mesma estrutura, alterando apenas informações como:

- Nome da empresa
- Cores
- Produtos
- Imagens
- WhatsApp
- Promoções

Sem necessidade de alterar a lógica principal do sistema.
